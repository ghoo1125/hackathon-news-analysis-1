01 堰塞湖形成與潰決災情

花蓮馬太鞍溪堰塞湖於2025年7月中旬形成，9月23日受樺加沙颱風豪雨影響發生潰決，洪水量相當於6,000個奧運標準游泳池。災害造成19人死亡、5人失蹤，光復鄉市區大面積淹沒，馬太鞍溪橋遭沖毀。崩塌量體比2009年小林村大10倍，屬於重大天然災害。

02 爆破爭議與專業評估

花蓮立委傅崐萁質疑中央未比照過去南投堰塞湖採取爆破處置。內政部長劉世芳於兩週前邀請前部長李鴻源組成專家團隊評估，結論認為爆破不可行且風險過高。專家強調此次堰塞湖位於深山，地形複雜，與過去案例情況不同，最終成功疏散7-8千人，降低傷亡。

03 災後重建與持續監測

國民黨團推動「災後重建特別條例」，初步匡列200億元重建預算。李鴻源提出六大遷村建議，並警告可能發生二次堰塞湖。10月和11月馬太鞍溪再度出現新堰塞湖，2,720戶進入紅色警戒。全台「鏟子超人」志工湧入協助清淤，但也發生「挖土機超人」不幸逝世的憾事，專家呼籲民眾勿貿然進入災區。

利害關係人立場

中央政府（內政部）

核心訴求

依法行政、專業評估優先、確保人民安全。

最新觀點陳述

  • • 專業團隊評估：兩週前邀請李鴻源組成專家團隊，經過詳細評估後認為爆破方案風險過高，可能造成更大災害，最終決定採取監測與疏散策略。
  • • 成功疏散7-8千人：在專家建議下提前部署疏散計畫，成功撤離7,000-8,000名居民，大幅降低傷亡人數。
  • • 持續監測應變：與林業保育署密切合作，透過衛星影像和空中勘查即時監測堰塞湖水位變化，確保預警系統運作正常。
  • • 重建預算支持：配合立法院推動200億元災後重建特別條例，協助花蓮縣政府進行災後復原工作。

水利專家（李鴻源）

核心訴求

科學決策、專業至上、長期防災規劃。

最新觀點陳述

  • • 反對爆破方案：堰塞湖位於深山，地形複雜且崩塌量體達小林村10倍，爆破可能引發更大規模崩塌，風險遠大於效益。與過去南投案例情況完全不同。
  • • 警告二次災害：馬太鞍溪上游仍有不穩定地質，可能再次形成堰塞湖。10月和11月已多次出現新堰塞湖，證實了預警的準確性。
  • • 提出遷村建議：針對高風險區域提出六大遷村建議，認為長期而言應將居民遷離危險地帶，而非持續投入資源維護不安全的居住環境。
  • • 呼籲志工勿冒險：多次強調「災區不是你想的那樣子」，專業救災需要訓練和設備。挖土機超人不幸逝世後，更強烈建議民眾不要貿然進入災區。

花蓮縣政府（徐榛蔚）

核心訴求

加速救災、協助重建、爭取中央資源。

最新觀點陳述

  • • 公開致歉：首度為堰塞湖洪災造成19死5失蹤公開致歉，表示「責無旁貸」，承認縣政府對災情負有責任。
  • • 協調地方救災：統籌地方救災資源，配合林業保育署紅色警戒發布疏散命令，協助受災居民安置工作。
  • • 持續監測警戒：11月多次新堰塞湖形成，2,720戶進入紅色警戒。縣府持續與中央和林保署合作，維持24小時監測機制。
  • • 推動災後重建：配合中央推動200億重建預算，規劃光復鄉、鳳林鄉等受災地區的基礎建設修復和居民安置方案。

國民黨立法院黨團（傅崐萁）

核心訴求

質疑中央作為、爭取重建預算、要求政府負責。

最新觀點陳述

  • • 質疑不爆破決定：認為中央應比照過去南投堰塞湖案例採取爆破處置，質疑為何此次不採取更積極的作為。對專家「爆破不可行」的結論持保留態度。
  • • 推動200億重建預算：優先推動「樺加沙風災暨花蓮光復堰塞湖潰決災後重建特別條例草案」，初步匡列200億元重建經費。
  • • 要求中央負責：認為中央政府應對災害應變負起更大責任，不應將責任推給地方政府或歸咎於天災不可抗力。
  • • 監督重建進度：持續監督災後重建預算執行，確保資源確實用於受災地區，不被挪作他用。

林業及自然保育署（花蓮分署）

核心訴求

科學監測、即時預警、提供決策依據。

最新觀點陳述

  • • 持續空中勘查：自7月堰塞湖形成以來，持續透過直升機進行空中勘查，記錄堰塞湖水位、蓄水量和壩體穩定度變化。
  • • 紅色警戒發布：根據監測數據評估潰決風險，適時發布紅色警戒。11月新堰塞湖形成後，立即通報縣政府啟動疏散機制。
  • • 蓄水量即時更新：9月23日潰決前監測到3千多萬噸蓄水，潰決後蓄水量降至15%。11月新堰塞湖最高達1,500萬噸，後降至20萬噸並解除警戒。
  • • 衛星影像追蹤：配合「翻轉地理教室」等單位，利用衛星影像追蹤全台堰塞湖狀況。目前馬太鞍溪上游仍有2處堰塞湖在監測中，1處已消失。
